📱 Nokos API Web

Interface untuk mengelola nomor dan OTP

📌 Format API Key: zel_xxxxxxxxxxxxxxxxx (contoh: zel_abc123def456)
🌐 API Base URL: https://zelapioffciall.dpdns.org/api/v1/nokos
Format: zel_ diikuti dengan huruf/angka (contoh: zel_abc123def456)
🔌 Test Koneksi:

📊 Last Numbers GET

Mendapatkan daftar nomor terakhir

Response: [[service, number, otp, time, country]]

Klik tombol untuk melihat hasil

⏳ Lifetime GET

Data lifetime numbers

Klik tombol untuk melihat hasil

📡 Feed GET

Feed OTP terbaru

Response: [{number, service, otp, time}]

Klik tombol untuk melihat hasil

📞 Request Number POST

Response: {success, number, id}

Klik tombol untuk request

🔐 Latest OTP GET

Response: {otps: {number: otp}}

Masukkan nomor dan klik tombol

♻️ Release Number POST

Response: {success, message}

Masukkan nomor dan klik tombol
